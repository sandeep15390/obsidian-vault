# Telugu Teacher Voice Agent

## Overview
A voice-based AI agent that teaches Telugu to English speakers whose first language is English.

## Goals
- Help native English speakers learn Telugu through conversational voice interactions
- Make Telugu language learning accessible and interactive

## Key Features
- Voice input/output for natural conversation practice
- English to Telugu translation and pronunciation guidance
- Lesson progression tailored to English speakers
- Cultural context and common phrases
